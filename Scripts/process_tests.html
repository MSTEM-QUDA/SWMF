Interpretation of Test Results

Notation used in the tables:
  • Red text shows what failed in a test.
  • Green text indicates that a test passed.
  • CAPITALIZED text shows results that have changed.
  • Click on the failed tests, the results or the log file links under the machine names for more information.
Scores shown below each day are the ratio of achieved and maximum possible total test results some weights for the machines and for the tests based on their relevance for certain projects (CCHM, CWMM, CRASH). The test results themselves are given a simple numeric value
  • 1.0 - passed
  • 0.5 - result
  • 0.1 - run
  • 0.0 - compile
The same scores are saved into the all_scores.txt files in the test result archive.
