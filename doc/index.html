SWMF MANUALS

  • User Manual in PDF format and in HTML format

    Introduction to the SWMF. The basic usage with example runs. Complete list of input commands with an index (in PDF format only).

  • Reference Manual in PDF format and in HTML format

    Contains the documentation of the source code and it also contains description of the makefiles and various scripts.

  • Physics of the Couplings in PDF format

    A brief description of the physics of the component couplings.

  • Testing Procedures in PDF format

    The testing procedures used for code verification and portability testing.

  • Maintenance Manual in PDF format

    The maintenance manual is mostly intended for developers.

  • Design Document in PDF format and in DOC format

    The design from a software engineering point of view.

  • Data Naming Standard in PDF format

    The data naming convention adapted by the developers of the SWMF in the Backus-Naur Format (BNF)
